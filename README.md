bin
===
A collection of scripts to manage my system/workflow/productivity.

Most of them are written in POSIX shell.

Installation
------------
To install *every* script, use git to clone this repository and then add it to
your [PATH][1]

[1]: https://katiek2.github.io/path-doc/

Or you can grab the individual scripts.
